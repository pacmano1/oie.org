Announcing: OIE Source Code Search - Now Available from [Diridium Technologies](https://diridium.com)

We’re excited to announce the release of **[OIE Source Code Search](https://github.com/diridium-com/oie-source-code-search)**, an open-source plugin for Open Integration Engine (OIE) that that provides grep-like search across all channel scripts, code templates, global scripts, and message templates - directly from the Administrator UI.

![Source Code Search UI](/images/news/source-code-search.png)

- **Full-text search** across all artifact types in a single query
- **Literal and regex** search modes, case-sensitive or insensitive
- **Scope control**  - search Channels, Code Templates, Global Scripts, and Message Templates independently
- **Channel scoping**  - search all channels, selected channels, or the current channel from the editor
- **Hierarchical results** with location breadcrumbs and match highlighting
- **Export** results as JSON (with metadata) or CSV
- **Non-modal dialog**  - search while you work

See the [Wiki](https://github.com/diridium-com/oie-source-code-search/wiki) for full documentation.
